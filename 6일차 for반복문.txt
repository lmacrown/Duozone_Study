배열 : 연속된 공간에 값을 나열하는 것. 각 값은 인덱스를 부여해놓은 자료구조.
type[] 참조변수 = null; // 
초기화 후 다시 선언시 기존의 데이터는 초기화
정적 코드 : 정확히 기록되어 값이 바뀌지 않는 코드
동적 코드 : 앱을 실행하면서 데이터를 기록해서 사용을 하는 코드
함수 : 논리적으로 만들기 위해 사용
다차원 배열에서 length는 행에 해당되는 값
score[1].length일경우 해당 열의 값을 표현
arraycopy(arr1, 0, arr2,0, arr1.length) : arr1값(길이만큼)을 arr2값에 대입
for(타입 변수 : 배열){ 실행문;}
enum타입 : 한정된 값을 가지는 타입
날짜 계산 : calendar 라이브러리 사용
지역변수가 될경우 전역변수로 선언하고 null로 초기화시킴
