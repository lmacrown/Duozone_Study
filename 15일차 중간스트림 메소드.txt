getAsDouble() : double형 데이터로 가져옴
BaseStream : 모든 스트림에서 사용할 수 있는 공통 메소드들을 정의
람다함수에서는 지역변수를 선언할수 없음
lines(경로, 언어셋) : 해당 경로에 있는 리소스 값을 해당 언어로 번역하여 저장
중간 스트림 : 중간 처리에서 요소를 정제(반복되는 자료 처리, 요소 정렬)
- distinct() : 요소의 중복 제거
- predicate<T> : 객체 T 조사
- filter() : 요소 필터링
매핑 : 스트림의 요소를 다른 요소로 변환하는 중간 처리 기능
- mapXxx() : 요소를 다른 요소로 변환한 새로운 스트림을 리턴
- boxed() : 객체로 생성함 
요소 정렬 : 요소를 순차적으로 정렬
- sorted
- comparable : 직접 구현하여 요소 정렬
- comparator : 직접 구현하지 못하고 객체 두개를 받아서 요소 정렬
루핑 : 스트림에서 요소를 하나씩 반복해서 가져와 처리하는 것
- peak() : 중간처리 메소드
- forEach() : 최종 처리 메소드
- accept() : 매개값을 처리하는 메소드
요소 조건 만족 여부 : 매칭하는 요소들이 특정 조건에 만족하는 지 여부 조사하는 최종 처리 기능
- allMatch() : 모든 값에 대한 비교
- anyMatch() : 하나라도 값에 해당하는 지 비교
- noneMatch() : 하나도 값에 해당 안되는지 비교
요소 기본 집계 : 카운팅, 합계, 평균값, 최대값, 최소값 등과 같이 하나의 값으로 산출하는 최종 처리 기능
Optional 클래스 : 단순히 집계값만 저장하는 게 아니라 집계값이 존재하지 않을 경우 디폴트 값을 설정
(0/0 같은 계산 불가한 것을 처리)
