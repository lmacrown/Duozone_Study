콘솔에서 cp를 안쓸려고 할 경우 : 환경변수에서 classpath에서 경로를 지정하여 실행
cmd에서 파일 jar 사용해서 자체 라이브러리 사용 하는 법
java -cp bin;C:\workspace\my_lib\dist\my_lib.jac app.Main
일반 라이브러리는 내부에 포함된 모든 패키지에 외부 프로그램에서의 접근이 가능
모듈 : 일부 패키지를 은닉하여 접근할 수 없게 가능
클래스에서 타클래스를 선언하여 사용 가능
class A { B b= new B();}
전이 의존 개체 : a에 module b를 할당하여 a모듈만을 호출하여도 모든 메서드 사용가능
module-info.java : 컴파일시 사용하는 파일
플러그인 : 연결하여 사용
transitive : 자동으로 의존적인 모듈들을 연결해줌
집합 모듈 : 여러 모듈을 모아놓는 것. 모듈 기술자에 전이 의존 설정만 함
리플랙션 : 실행 도중에 클래스를 로딩하여 타입을 검사하고 구성 멤버를 조사하는 것(클래스 관리)
=>RTTI(Run Time Type Information)
문자열 숫자 : 종속성

예외 : 잘못된 사용 또는 코딩으로 인한 오류 => 예외 처리 필수
일반 예외 : 컴파일러가 예외 처리 코드 여부를 검사하는 예외
실행 예외(Runtime Exception) : 컴파일러가 예외 처리 코드 여부를 검사하지 않는 예외
try{ 예외 체크 } catch{ 예외 발생시 수행 } finally { 필수 실행 }
printStackTrace() : 예외가 발생한 에러를 호출해줌
특별한 예외처리가 앞에 있어야함 => 그러지 않으면 특별한 예외처리를 안하고 일반 예외처리만 함
리소스 : 데이터를 제공하는 객체. 사용하고 나서 닫아야함
throws : 예외를 호출한곳으로 떠넘김
