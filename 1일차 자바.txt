자바는 자동으로 변환시켜줌
자바는 한번 만들어지면 어디서든 사용가능
RestAPI : URI, 행위(verb), 표현으로 구성.
유니폼 인터페이스, 무상태성, 캐시가능, 자체 표현 구조, 클라이언트 서버 구조, 계층형 구조
exe, msi, com : 기계어 파일이며 바이러니 파일
bat : 텍스트 파일
컴퓨터 시스템은 공용보다 사적인 곳을 먼저 찾음
환경변수에서 path설정할때 %를 쓰면 변수를 지정한다는 뜻
javac -d bin 을 사용하면 해당하는 파일에 java클래스를 만듬
java -cp bin 을 사용시 해당 파일 실행
sysout 입력후 ctrl+spacebar => 자동완성
bin : 바이너리 파일(실행 파일)
