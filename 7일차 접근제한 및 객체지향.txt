public : 외부에서 접근 가능
default : 같은 패키지 내에서만 접근 가능
protected : 같은 패지키 또는 자식 객체에서만 접근 가능
private : 클래스에서만 사용. 메서드를 통해서만 접근가능. 외부에서 객체 생성 불가
import : 다른 패키지에 있는 클래스 사용
ctrl shift O : import를 한번에 다 해줌

캡슐화(은닉화) : 데이터를 외부에 숨김. 
객체의 필드를 외부에서 마음대로 읽고 변경할 경우 객체의 무결성이 깨질수 있음
일반 멤버 : 필드, 일반 메서드 
클래스 멤버 : 정적필드(static)-공유변수, 정적 메소드(공유할수 있는 메서드)
결합력은 낮을수록 좋고 응집력은 높을수록 좋은 코드이다.
객체지향 프로그램을 만들기 위해 SPC를 잘 지켜야한다.
코드는 소설같으면서 잘 다듬어야한다.
